## Key accomplishments

1. **Accuracy:** retrained on a 154-book verified-clean corpus → hold-out macro-F1 **0.7367** (vs v1 0.3235, +41pp, p=0.0010).
2. **Calibrated, explainable predictions:** top-N with real `predict_proba` (Brier 0.0481) + a ~15 ms "why this genre?" panel (nearest books, topology-vs-vocabulary contribution, driving words).
3. **Cache lineage hardening:** every content-addressed artifact is keyed by `corpus_hash` + `w2v_model_sha256`, so a retrain can never serve stale precomputes.
4. **Two front-end generations:** Phase 10/11 brought theming + onboarding to the indigo UI, then Phase 12 replaced the whole front end with the editorial "Reading Room."
5. **Honest validation:** author-leakage surfaced (CEXP-04) and disclosed in-app rather than hidden — the macro-F1 is presented as an upper bound.

## Milestone summary — decisions, gaps, carry-overs

**Notable decisions:** H₂ deferred to v3 (degenerate/empty + O(n⁴) cost); ship-with-disclaimer for author leakage (D-31/D-33); explainability via nearest-neighbour + zero-ablation (no synchronous SHAP); light default + paper/accent Tweaks (reversing the earlier dark-default); R3F WebGL scatter retained (reskinned) over the handoff's SVG plates.

**Known gaps carried into v2.1** (documented caveats, not phase gaps — see `REQUIREMENTS.md` §"v2.1 Carry-over"):
- **V21-01** Author-leakage gap (CEXP-04 BLOCKED) — 45pp GroupKFold-by-author gap; shipped with disclaimer.
- **V21-02** α weight miscalibrated for the v2 corpus — LOOCV (2026-05-30): location-only 0.7682 > shipped 70/30 0.6887 ≫ topology-only 0.2053; α=0.7 (tuned on v1) underperforms location-only by ~8pp. Re-sweep α on the v2 hold-out protocol.
- **V21-03** Phase 9's 7 human-UAT items still `partial` (automated verification passed; live walkthrough pending).
- **V21-04** Per-book persistence not cached for all books (Topology tab is region-keyed as a result).
