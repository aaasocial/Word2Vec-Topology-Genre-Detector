# Phase 3 — UI Design Contract

> Visual and interaction contract for the frontend core and 3D visualization phase. Generated by gsd-ui-researcher, verified by gsd-ui-checker.

---

## Design System

| Property | Value |
|----------|-------|
| Tool | shadcn |
| Preset | dark mode, default radius, neutral base (initialize via `npx shadcn@latest init` with `--style default`) |
| Component library | Radix UI (via shadcn/ui) |
| Icon library | lucide-react |
| Font | `Inter` for UI labels; `JetBrains Mono` for data values (TF-IDF weights, coordinates, word tokens) |

---

## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, inline badge padding, tooltip internal padding |
| sm | 8px | Between label and value pairs, compact list item padding |
| md | 16px | Default element spacing, sidebar section gaps, control group padding |
| lg | 24px | Between sidebar sections (e.g., upload zone to genre dropdown) |
| xl | 32px | Sidebar outer padding |
| 2xl | 48px | Not used this phase |
| 3xl | 64px | Not used this phase |

Exceptions: Touch targets for sidebar controls minimum 40px height (desktop-only app, no 44px mobile requirement).

---

## Typography

| Role | Size | Weight | Line Height | Font |
|------|------|--------|-------------|------|
| Body | 14px | 400 (regular) | 1.5 | Inter |
| Label | 12px | 400 (regular) | 1.4 | Inter |
| Heading | 18px | 600 (semibold) | 1.2 | Inter |
| Mono | 13px | 400 (regular) | 1.4 | JetBrains Mono |

**Usage rules:**
- Body (14px): sidebar descriptions, tooltip body text, search results list
- Label (12px): control labels ("Genre", "Projection", "Book"), axis labels, keyboard shortcut hints, progress step names
- Heading (18px): sidebar panel titles ("Controls", "Upload", "Search Results", "Classification Result")
- Mono (13px): word tokens in tooltips and detail panels, TF-IDF numeric values, confidence percentages, coordinate values

### Genre Color Palette (VIZ-11)

10 genres require 10 visually distinct, colorblind-safe hues on dark backgrounds. These are used ONLY for scatter plot point coloring and genre legend indicators.

| Genre | Hex | Tailwind-ish name |
|-------|-----|-------------------|
| romance | `#F472B6` | pink-400 |
| mystery | `#60A5FA` | blue-400 |
| western | `#FB923C` | orange-400 |
| fantasy | `#A78BFA` | violet-400 |
| scifi | `#34D399` | emerald-400 |
| horror | `#F87171` | red-400 |
| historical | `#FBBF24` | amber-400 — NOTE: when uploaded book is active, historical dims to `#D97706` (amber-600) to avoid collision |
| literary | `#2DD4BF` | teal-400 |
| adventure | `#FB7185` | rose-400 |
| gothic | `#C084FC` | purple-400 |

**Brightness encoding (VIZ-03):** Genre color is the base hue. TF-IDF weight maps to emissive intensity (0.0 = dim/near-invisible at 10% opacity, 1.0 = full brightness at 100% opacity). Points outside the selected genre render at 8% opacity with no emissive.

**Selected point (VIZ-07):** White ring outline (`#FFFFFF`, 2px equivalent screen-space) around the clicked point. Persists through camera movement until Esc or click elsewhere.

## Interaction Contracts

### Hover States

| Element | Hover behavior |
|---------|---------------|
| Scatter point | Point grows 1.5x, emissive increases. Tooltip appears after 0ms delay. |
| Sidebar control | Background lightens to `#1A1A25` (inputs already at this color get `#222230`) |
| Genre legend entry | Text brightens to `#F5F5FF` |
| Button (outline) | Border color shifts to `#6366F1`, text to `#818CF8` |
| Button (default) | Background lightens to `#818CF8` |
| Dropdown option | Background `#1A1A25` |

### Click States

| Element | Click behavior |
|---------|---------------|
| Scatter point | White ring appears. Detail panel opens in sidebar. Previous selection cleared. |
| Projection tab | Instant switch. All points animate to new positions over 600ms (lerp). |
| Genre dropdown | TF-IDF brightness recalculates for new genre. Transition: 300ms opacity fade on all points. |
| Book slider | Brightness shifts per-book. 200ms debounce then 300ms transition. |
| Reset camera | Camera tweens to default position/rotation over 500ms. |
| 2D/3D toggle | Points animate Z coordinate to 0 (2D) or restore (3D) over 600ms. |
| Upload drop zone | Opens native file picker for .txt files. |
| Search result item | Selects that point in scatter (same as clicking the point). |
| Genre legend entry | Sets genre dropdown to that genre. |
| "View in Scatter" button | Camera pans to uploaded book centroid, 800ms tween. |

### Drag-and-Drop Feedback

| State | Visual |
|-------|--------|
| File dragged over window | Drop zone border pulses `#6366F1`. "Drop .txt file here" text brightens. |
| File dragged over drop zone | Solid `#6366F1` border, `rgba(99,102,241,0.08)` background fill. |
| File dropped (valid) | Drop zone replaced by progress stepper within 100ms. |
| File dropped (invalid type) | Red flash on border (`#EF4444`), error text: "Only .txt files accepted" for 3 seconds. |
| File dropped (too large) | Red flash on border, error text: "File exceeds 5MB limit" for 3 seconds. |

### WebSocket Progress Display

| Event | UI update |
|-------|-----------|
| Connection opened | First step dot activates (pulse animation). |
| Step transition message | Previous step marked complete (green check). Next step dot activates. Progress bar advances proportionally (1/6 per step). |
| Error message | Current step shows red X. Error text appears below. "Retry" button appears. |
| Completion message | All steps green. Progress bar fills to 100%. After 500ms, classification result panel fades in below. |
| WebSocket disconnect | If unexpected: "Connection lost. Retrying..." message with spinner. Auto-retry 3 times at 2s intervals. |

### Transition Animations

| Transition | Duration | Easing |
|------------|----------|--------|
| Projection switch (point positions) | 600ms | ease-out (cubic-bezier 0.33, 1, 0.68, 1) |
| Genre change (brightness shift) | 300ms | ease-in-out |
| Camera reset | 500ms | ease-out |
| Camera pan to uploaded book | 800ms | ease-in-out |
| 2D/3D toggle (Z collapse/expand) | 600ms | ease-out |
| Sidebar collapse/expand | 300ms | ease-out |
| Tooltip appear | 0ms (instant) |
| Tooltip disappear | 0ms (instant) |
| Detail panel appear | 150ms | fade-in |
| Classification result appear | 500ms | fade-in |
| Upload error flash | 300ms | flash then fade |
| Keyboard shortcut hint fade | 1000ms | fade-out |
