# Phase 06 Plan 05: BUG-05 — Cache key includes corpus_hash and w2v_model_sha256

## Objective

Close the latent v1 cache-key footgun before Phase 8 retrain by extending the content-addressed cache key to include the corpus manifest hash (`corpus_hash = sha256(corpus/books.yaml)`) and the Word2Vec model hash (`w2v_model_sha256 = sha256(data/models/word2vec_w{window}.model)`). Eagerly flush `data/cache/` in the same commit so v1 stale-cache artifacts cannot leak across the v2 boundary. Pin SVM training-data lineage as defense-in-depth (D-25).

- This is **the LAST plan in phase 6** — D-26 ordering ensures the eager cache flush is not redone after later work.

Modify `backend/cache/store.py`:

```python
def cache_key(
    step_name: str,
    params: dict,
    *,
    corpus_hash: str,
    w2v_model_sha256: str,
) -> str:
    """Generate deterministic cache key from step name, params, and lineage.

    Lineage (corpus_hash + w2v_model_sha256) is mandatory: a corpus change or
    Word2Vec retrain MUST invalidate every downstream artifact (PITFALLS §1).
    Keyword-only arguments force every call site to pass them explicitly —
    no silent defaults that would re-introduce the v1 footgun.

    Args:
        step_name: stable identifier for the pipeline step.
        params: step-specific parameters (order-invariant via sort_keys).
        corpus_hash: sha256 of corpus/books.yaml content.
        w2v_model_sha256: sha256 of the Word2Vec model file in use.

    Returns:
        Hex sha256 of the canonicalized payload.
    """
    canonical = json.dumps(
        {
            step_name: params,
            '__corpus_hash__': corpus_hash,
            '__w2v_model_sha256__': w2v_model_sha256,
        },
        sort_keys=True,
        default=str,
    )
    return hashlib.sha256(canonical.encode()).hexdigest()
```

Also add a `lineage.py` helper module with:

```python
# backend/cache/lineage.py
import hashlib
from pathlib import Path

def file_sha256(path: Path) -> str:
    """Stream-hash a file. Returns hex digest."""
    h = hashlib.sha256()
    with open(path, 'rb') as f:
        for chunk in iter(lambda: f.read(65536), b''):
            h.update(chunk)
    return h.hexdigest()

def corpus_hash() -> str:
    """sha256 of corpus/books.yaml — the canonical corpus manifest."""
    root = Path(__file__).resolve().parents[2]
    return file_sha256(root / 'corpus' / 'books.yaml')

def w2v_model_sha256(window: int) -> str:
    """sha256 of the Word2Vec model file for the given window."""
    root = Path(__file__).resolve().parents[2]
    return file_sha256(root / 'data' / 'models' / f'word2vec_w{window}.model')
```
