### Heatmap Color Scale Specification

The persistence image heatmap uses the **plasma** colorscale with 256 discrete color stops. The scale is linear from data minimum to data maximum. When two genres are compared (COMP-02), the color scale range is computed from the combined min/max of both datasets, ensuring visual comparability.

Heatmap cell rendering: Canvas2D `fillRect` per cell (persistence image is M x M grid, typically 20x20 to 50x50). No anti-aliasing between cells -- sharp pixel boundaries.

## Interaction Contracts (Phase 4 Additions)

### Tab Switching

| Transition | Behavior |
|------------|----------|
| Tab click | Active tab content replaces previous instantly (no crossfade). Sidebar persists across tabs. |
| Scatter to Topology | Canvas area splits into two panels (heatmap + VR viewer). Sidebar genre/book selection persists. |
| Topology to Scatter | Two panels merge back to full canvas. Camera position resets to scatter default. |
| Any to Compare | Compare mode activates within Scatter tab. Second genre picker appears in sidebar. |

### Brushing and Linking (TOPO-07)

| Action | Effect |
|--------|--------|
| Change genre in sidebar dropdown | Scatter updates TF-IDF brightness. Persistence heatmap reloads for new genre. VR viewer reloads edges for new genre. |
| Change book in sidebar slider | Same as genre change but at book granularity. All panels update simultaneously. |
| Hover point in VR viewer | Same tooltip as scatter (component 13). No cross-highlight in other panels (too noisy). |
| Click point in VR viewer | Same detail panel behavior as scatter click (component 14). |

### Epsilon Slider Interaction

| Action | Effect |
|--------|--------|
| Drag slider | Edges with `eps_birth <= current_epsilon` become visible. Edges above threshold hidden. All filtering happens in browser -- no server calls. |
| Reach a birth event | New edges appear in `#FACC15` (yellow highlight). They remain highlighted for 500ms, then fade to default edge color `#4A4A5A` over 300ms. |
| Reach a death event | Dying edges flash `#FACC15` for 500ms, then disappear. |
| Release slider | No additional behavior. Current state persists. |

### Settings Drawer Interaction

| Action | Effect |
|--------|--------|
| Open drawer | Drawer slides in from right (300ms). Backdrop appears. Main content remains visible but dimmed. |
| Change slow-tier param | Amber "Parameters changed -- Recompute" badge appears. No computation triggered yet. |
| Click Recompute | Badge disappears. Button enters loading state. Recomputation overlay appears on canvas. When complete: overlay fades out (300ms), new data replaces old. |
| Change very-slow param | Confirm dialog opens immediately (no badge accumulation). |
| Confirm retrain | Dialog closes. Full canvas overlay with "Retraining model..." All interactions disabled except cancel. On completion: full page data refresh. |
| Cancel retrain | Revert parameter to previous value. Resume normal state. |

### Compare Mode Interaction

| Action | Effect |
|--------|--------|
| Activate compare | Second genre picker appears (200ms slide-down). Sidebar persistence heatmaps appear below. |
| Select second genre | Scatter renders both genres simultaneously (dual brightness). Sidebar shows stacked heatmaps. |
| Change either genre | Both scatter and both heatmaps update. Color scale recomputed from combined data. |
| Deactivate compare | Second picker and heatmaps slide out (200ms). Scatter returns to single-genre mode. |

### Pipeline Explanation Interaction

| Action | Effect |
|--------|--------|
| Open | Overlay fades in (200ms). Content appears. |
| Navigate steps | Content crossfades (200ms). Step indicator updates. |
| Step 4 mini VR | User can drag the mini epsilon slider within the explanation. Edges appear/disappear in the mini viewer. |
| Close | Overlay fades out (200ms). User returns to previous app state unchanged. |

### Export Interaction

| Click Export PNG (scatter) | Browser download dialog immediately. File downloads. Button shows check icon for 2s then reverts. |
| Click Export PNG (heatmap) | Same behavior. |
